Insurance Policy & Claims Analysis: Prism Insurance Pvt. Ltd.
Project Overview
This project provides a comprehensive analysis of the financial health and operational efficiency of Prism Insurance. By analyzing policy distribution, claim statuses, and loss ratios, the dashboard identifies key areas where the company is losing revenue and suggests actionable improvements.
Tech Stack
•	Database: MS SQL Server
•	Visualization: Power BI
•	Data Source: CSV (Insurance Records)
Key Analysis Areas
1. Financial Performance (Loss Ratio)
•	Discovered a significant financial imbalance with a Total Loss Ratio of 282.96%.
•	Identified Travel Insurance as the highest loss-making category, followed by Health.
•	Determined that Home Insurance is the only category experiencing minimal losses.
2. Operational Efficiency
•	Claim Status Tracking: Found that only 33.85% of policies are settled, while nearly 44% are rejected.
•	Bottleneck Identification: Identified a 22.6% backlog in pending policies, specifically in the Travel segment, suggesting a need for faster processing workflows.
3. Customer Demographics
•	Age Groups: The Adult group (25-60) represents over 50% of the policyholders across all types.
•	Gender Trends: Females show a higher preference for Travel and Auto insurance, while Males opt more for Health insurance.
Strategic Suggestions Included
•	Premium Adjustments: Immediate recommendation to raise premiums for high-volume, high-loss Travel policies.
•	Targeted Marketing: Increase awareness for Life Insurance among males and General Insurance for the 18-25 age group.
•	Process Optimization: Implement faster workflows for pending claims to improve customer retention and stabilize the financial outlook.
Repository Contents
•	SQLQuery-importing data.sql: SQL script for database setup and data validation.
•	PRISM INSURANCE PVT.LTD. Report.pbix: Interactive Power BI dashboard.
•	Observations,Insights and Suggestions.docx: Detailed breakdown of the analytical findings.
